# 브랜드 이미지 안내

아래 파일명으로 이 폴더(`public/images/`)에 넣어주세요. 파일이 없으면
사이트는 자동으로 고딕 텍스트 로고("hvving")로 폴백됩니다.

| 파일명 | 어떤 이미지 | 쓰이는 곳 |
| --- | --- | --- |
| `hvving-chrome.png` | 은색 크롬 가로 워드마크 | 메인 상단 대형 로고 |
| `hvving-vertical.png` | 세로 고딕 가시체 로고 | 메인 좌측 세로 컬럼 (데스크탑) |
| `hvving-bw.png` | 흑백 포스터화 워드마크 | (예비) |
| `hvving-chrome-blue.png` | 파란 크롬 워드마크 | (예비) |

- 배경은 검정 단색입니다(에셋 불필요).
- PNG는 **투명 배경**으로 저장하면 검정 위에 자연스럽게 얹힙니다.
- 참고: 채팅에 붙여넣은 이미지는 파일로 추출이 안 됩니다. 이 폴더에
  직접 파일을 커밋하거나, 접근 가능한 URL을 주시면 받아서 넣어드릴게요.
